# DevOps Git Project
## Overview
This project demonstrates Git best practices for a DevOps pipeline.

## Branch Strategy
- main → production
- dev → development
- feature/* → feature branches

## Workflow
1. Work in feature branch
2. Merge feature branch into dev via Pull Request
3. Test changes
4. Merge dev into main
5. Tag releases
